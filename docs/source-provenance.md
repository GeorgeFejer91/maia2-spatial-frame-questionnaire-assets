## MAIA-2

English MAIA-2 item text follows the official UCSF MAIA-2 2018 source document. German MAIA-2 item text follows the validated German 37-item MAIA-2 from Eggart, Todd, and Valdes-Stauber (2021), also available from UCSF as `MAIA-2_German.pdf`.

The German source PDF is stored locally:

```text
references/MAIA2_German_Eggart-2021_S1.pdf
```

The UCSF English source PDF URL was verified, but direct local download was blocked by the UCSF browser challenge during repository creation. The URL remains listed in `docs/citations.md`.
